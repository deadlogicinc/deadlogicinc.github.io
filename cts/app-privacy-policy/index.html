Location Data (Android-only)

The Android CTS app requires high-precision location access for BLE scans to work. This is a platform limitation—your location data is never used directly by CTS.

Starting from Android 6.0, the Android operating system requires apps to be granted location access in order to scan for BLE devices. Starting from Android 10, this requirement became stricter—high-precision location access is now required for BLE scans. Android 12 and newer versions of Android let developers use a different set of Bluetooth permissions, but by omitting the location permission, some BLE beacons may not show up, so we have decided to keep location access as a required permission to use the app.
